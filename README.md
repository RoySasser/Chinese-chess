# 中国象棋 (Chinese Chess) - 前端版

这是一个纯前端实现的中国象棋游戏，支持红黑双方对局。

## 目录结构
- `index.html`: 游戏主页面
- `style.css`: 样式文件，负责棋盘绘制和棋子样式
- `app.js`: 游戏逻辑脚本，处理走棋规则和交互

## 运行方式

### 方法 1：直接打开
直接在浏览器中打开 `index.html` 文件。
建议使用 PC 端浏览器或平板模式以获得最佳体验。

### 方法 2：静态服务器（可选）
如果您遇到本地文件访问限制，可以使用静态服务器运行。
例如，使用 Python 3：
```powershell
python -m http.server 8000
```
然后在浏览器访问：[http://localhost:8000](http://localhost:8000)

## 🎮 玩法指南

1. **基本规则**: 遵循标准中国象棋规则（马走日、象走田、炮翻山等）。
2. **操作**:
   - **选中棋子**: 点击己方棋子（高亮显示）。
   - **移动/吃子**: 点击目标位置或敌方棋子。如果目标合法，棋子将移动；如果是敌方棋子，则执行吃子。
   - **重新选择**: 如果想换个子走，再次点击另一个己方棋子即可。
3. **回合制**: 
   - 🟥 **红方先行**。
   - 顶部会显示当前回合方。
4. **胜利条件**: 吃掉对方的 **帅/将** 即可获胜。

## 特性
- 完整的走棋规则验证（包括蹩马腿、塞象眼）。
- 楚河汉界限制（象不过河、卒过河后横走）。
- 图形化棋盘与棋子。
